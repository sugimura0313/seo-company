# SEO・WordPress 知識ベース

## SEO基本戦略

### キーワード選定の原則
- **ロングテール優先**: 月間検索数100〜1,000のKWで勝負（競合が弱い）
- **検索意図の4分類**: 情報収集型 / 比較検討型 / 購買型 / ナビゲーション型
- **KGR（Keyword Golden Ratio）**: allintitle検索数 ÷ 月間検索数 < 0.25 が狙い目
- **LSIキーワード**: 共起語・関連語を本文に自然に含める

### 記事構成の鉄則
- タイトルに数字・年月・「方法」「コツ」「完全ガイド」を入れる
- リード文で「誰向け」「何がわかる」「読む価値」を30秒で伝える
- H2は検索意図に直接答える（ユーザーの疑問＝見出し）
- まとめ+CTAは必須（次の行動を明示）

### 内部リンク設計
- **サイロ構造**: カテゴリページ（柱記事）→ 個別記事（クラスター）
- 柱記事から全クラスターへリンク、クラスターから柱記事へリンク
- アンカーテキストはKWを含む自然な文言

## WordPress知見

### 推奨プラグイン（SEO特化）
| プラグイン | 用途 |
|------------|------|
| Rank Math / Yoast SEO | メタ設定・サイトマップ・構造化データ |
| WP Rocket / LiteSpeed Cache | ページ速度最適化 |
| Smush / ShortPixel | 画像圧縮 |
| Schema Pro | 構造化データ強化 |
| Broken Link Checker | 内部リンク管理 |

### Core Web Vitals 改善チェックリスト
- [ ] LCP < 2.5秒（画像最適化・CDN）
- [ ] FID < 100ms（JS遅延読み込み）
- [ ] CLS < 0.1（画像にwidth/height指定）
- [ ] TTFB < 800ms（サーバー・キャッシュ）

### 構造化データ（必須）
- Article / BlogPosting
- BreadcrumbList
- FAQPage（Q&Aセクションがある記事）
- LocalBusiness（地域SEO案件）

## 競合分析の手順
1. 狙うKWでGoogleトップ10を確認
2. 上位3記事の文字数・見出し構成・内部リンク数を調査
3. 自社記事は「上位記事より詳しい・わかりやすい・新しい」を目指す
4. 上位記事にない一次情報（体験談・事例・独自調査）を追加

## 想定支援事例（記事の一次情報として使用すること）

記事を書く際は、以下の想定事例を自然な形で盛り込み、E-E-A-T（経験・専門性・権威性・信頼性）を高めること。
「弊社が支援したケースでは〜」「実際の対応事例として〜」という表現で使用してよい。

### MEO支援事例

**事例A：横浜市の整骨院**
- 課題: Googleマップで同エリア内5位以下。新規患者が月10人程度
- 施策: GBPの基本情報修正・写真20枚追加・投稿を週2回・口コミ返信の徹底
- 結果: 3ヶ月でローカルパック（上位3枠）に安定表示。新規患者が月28人に増加（約2.8倍）
- ポイント: カテゴリを「整骨院」から「柔道整復師」に変更したことが順位改善のきっかけ

**事例B：大阪市の美容室（個人経営）**
- 課題: GBP未設定のまま2年営業。ネット経由の予約ゼロ
- 施策: GBP新規開設・写真30枚・メニュー価格登録・QRコードで口コミ依頼
- 結果: 開設2ヶ月で口コミ18件獲得。「梅田 美容室」で地図表示されるように
- ポイント: 口コミ返信を48時間以内に行うルールを設けたことで返信率100%を達成

**事例C：名古屋市の歯科医院**
- 課題: 競合が多く検索上位に入れない。広告費が月15万円かかっていた
- 施策: MEO最適化＋サービス項目（矯正・ホワイトニング等）の詳細登録
- 結果: 「名古屋 歯科 矯正」でローカルパック入り。広告費を月5万円に削減
- ポイント: 施術別のサービスページをHP内に作り、GBPからリンクしたことが効果的

### 士業Web集客事例

**事例D：東京都の税理士事務所（独立2年目）**
- 課題: ホームページはあるが問い合わせが月0〜1件。新規顧客獲得に苦労
- 施策: WordPress再構築・SEO記事30本・「フリーランス 確定申告 代行」等のKWで上位狙い
- 結果: 6ヶ月で月間PV 3,200。問い合わせ月8件に。顧問契約5社を新規獲得
- ポイント: 「税理士 費用 相場」より「フリーランス 税理士 いくら」などニッチKWが刺さった

**事例E：埼玉県の行政書士（開業1年目）**
- 課題: 開業直後でブランドゼロ。競合の大手事務所に埋もれていた
- 施策: 「建設業許可 埼玉 申請方法」など地域×業種の特化記事を集中制作
- 結果: 4ヶ月で「埼玉 建設業許可 行政書士」で1位表示。月3〜5件の安定受注
- ポイント: 地域名+業種に絞ることで大手と戦わないニッチポジションを確立

### IT補助金支援事例

**事例F：神奈川県の工務店（従業員8名）**
- 課題: 紙ベースの見積書・発注管理。DX化したいが予算がない
- 施策: IT導入補助金を活用したkintone導入支援。申請書類の作成サポート
- 結果: 補助率3/4で導入費用150万円のうち112万円が補助。自己負担38万円でDX化達成
- ポイント: 補助金申請の締め切り1ヶ月前から動き始めたことで余裕を持って対応できた

**事例G：大阪府の美容サロン（個人事業主）**
- 課題: ホームページが古く予約フォームもない。小規模事業者持続化補助金を活用したい
- 施策: 補助金申請（上限50万円）でWordPressサイトをフルリニューアル
- 結果: 採択率アップのポイント（計画書の書き方）を指導し、無事採択。ネット予約導入後、予約数が月40件増
- ポイント: 「販路開拓」の観点でHP制作の必要性を計画書に明記したことが採択のカギ

### WordPress保守・トラブル事例

**事例H：ECサイト（アパレル）のハッキング対応**
- 状況: ある朝サイトが真っ白に。調査すると不正ファイルが300個以上埋め込まれていた
- 原因: 3年間プラグインを一度も更新していなかった。WooCommerceの古いバージョンが狙われた
- 対応: 感染ファイル全削除・WordPressクリーンインストール・WordFence導入・2段階認証設定
- 復旧時間: 発見から8時間で復旧
- 教訓: 月1回のプラグイン更新と週1回の自動バックアップがあれば防げた案件

**事例I：士業事務所の表示速度改善**
- 状況: PageSpeed InsightsでモバイルスコアがF評価（32点）。直帰率が78%
- 原因: 未圧縮画像が計45MB・不要プラグイン23個・キャッシュ未設定
- 対応: Smushで画像圧縮・不要プラグイン削除・WP Rocket導入・CDN設定
- 結果: モバイルスコア32点→71点に改善。直帰率78%→54%に低下。問い合わせ数1.4倍
