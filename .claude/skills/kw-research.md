# キーワードリサーチ

## 使い方
`/kw-research [テーマ]`

例: `/kw-research 転職 30代`

## 実行内容
1. 指定テーマの関連KWを30個以上リストアップ
2. 検索意図（情報収集型/比較型/購買型）に分類
3. 競合の強さ（弱/中/強）を推定
4. 優先度順に並べ替え（最初に狙うべきKW上位5つを★マーク）
5. `tasks/keywords-[テーマ]-[日付].md` に保存

## 出力フォーマット
```
## [テーマ] キーワードリスト - [日付]

### ★ 優先5KW（今すぐ記事にすべき）
| KW | 検索意図 | 競合強度 | 推定月間検索数 |
|----|---------|---------|--------------|
| ...| ...     | 弱      | ...          |

### その他候補KW
...
```
