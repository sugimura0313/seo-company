# 記事執筆

## 使い方
`/write-article [キーワード]`

例: `/write-article 転職 30代 未経験`

## 実行内容
1. 検索意図を分析（このKWで検索する人が本当に知りたいこと）
2. 競合上位3記事の構成を調査・参考にしつつ差別化ポイントを設定
3. 記事構成（タイトル・H2/H3・リード文・まとめ）を作成 → **社長承認を求める**
4. 承認後、本文を執筆（最低3,000字、独自性重視）
5. メタディスクリプション（120字以内）を作成
6. 内部リンク候補を3本以上提示
7. `clients/[client-name]/articles/` または `articles/draft/` に保存

## WordPress投稿チェックリスト
- [ ] タイトルにKWが含まれている
- [ ] メタディスクリプションが設定されている
- [ ] アイキャッチ画像がある
- [ ] カテゴリ・タグが設定されている
- [ ] 内部リンクが3本以上ある
- [ ] 見出しにKWが含まれている（H2に少なくとも1つ）
